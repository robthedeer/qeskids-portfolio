# QESKIDS Portfolio & Impact Archive (2020–2025)

This repository is a comprehensive archive of my work as **Founder & CEO of QESKIDS**, a multi-center technology education enterprise in Moscow, Taldom, and Dmitrov, Russia.

It contains:
- 💻 Student projects in Web Development, Python, and JavaScript
- 🎨 3D design work created in Blender
- 🤝 Partnership documentation with global EdTech organizations
- 📸 Photos and videos documenting our social impact
- 🏆 Certificates and awards from Olympiads and bootcamps
